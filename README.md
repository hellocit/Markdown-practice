## 2.リスト
[-]でできる．タブを用いることで入れ子構造にできる．
- Hoge
    - hoge
    - hogehoge
- Fuga
    - fuga
    - fugafuga
単純に箇条書きするだけなら[*]でもできる．
* hoga
* hoge

## 3.チェックボックス

* [] no
